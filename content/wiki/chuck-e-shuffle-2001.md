***Chuck E. Shuffle*** is a generic live show released in September 2001 for Studio C stages. This was a re-edit of {{< wiki-link "Chuck E. Shuffle" >}}, which would play through November 2002. 

## Summary

At the beginning of the live show, the ‘Chuck E. Live’ logo from {{< wiki-link "Chuck E. March 2001" >}} appears. The rest of the live show uses the {{< wiki-link "January 2001 Show" >}} logo.

- This live show would be included in Studio C discs up until the release of {{< wiki-link "Chuck E. Shuffle 2006" >}}, though it was not playable as the programming was not included with any Studio C floppies after 2002.
