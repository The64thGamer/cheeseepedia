The animatronic archival website, Cheese-E-Pedia! Previously hosted through Miraheze using the MediaWiki platform, this website is now hosted through GitHub using Hugo.

You can refer to our [Contributor Home](https://cheeseepedia.org/meta/contributorhome/) for more information about this wiki, including how to edit pages using DecapCMS (which creates pull requests automatically for our review) or how to make a local copy of the site to test changes, compile locally, and submit pull requests manually.
